# Ejercicio 46 - Convertir un String a Entero

Crea una función que reciba un string y lo retorne como un número entero.

---

## 🧪 Ejemplos

```javascript
stringInt("6") ➞ 6

stringInt("1000") ➞ 1000

stringInt("12") ➞ 12
```

## 📝 Notas

- Todos los números serán enteros y positivos.
- No olvides retornar el resultado.
