# Ejercicio 47 - Área de un Rectángulo

Crea una función que calcule el área de un rectángulo.  
Si los argumentos no son válidos, la función debe retornar `-1`.

---

## 🧪 Ejemplos

```javascript
area(3, 4) ➞ 12

area(10, 11) ➞ 110

area(-1, 5) ➞ -1

area(0, 2) ➞ -1
```

## 📝 Notas

- No olvides retornar el resultado.
