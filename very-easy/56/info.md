# Ejercicio 56 - Invertir un Array

Escribí una función que reciba un array y lo retorne invertido (en orden inverso).

---

## 🧪 Ejemplos

```javascript
reverse([1, 2, 3, 4]) ➞ [4, 3, 2, 1]

reverse([9, 9, 2, 3, 4]) ➞ [4, 3, 2, 9, 9]

reverse([]) ➞ []
```

## 📝 Notas

- No olvides retornar el resultado.
